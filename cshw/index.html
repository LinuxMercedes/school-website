Parser for Matt Buechler's CS153 webpages. Just give it the URL for the homework webpage that you want to convert. It parses the webpage, creates a code file for each code file Buechler wants and puts in skeletal comments to make documenting easier. Edit the config variables to make life even easier for you.

Please let me know if you have any problems and/or have fixed a bug and/or have added a feature. My apologies for my terrible documentation; that will be forthcoming.
